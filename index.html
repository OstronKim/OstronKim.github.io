Visualization and data mining of the top Spotify songs the last decade

An application with interactvie visualization of the top Spotify songs the last decade, each song consisting of 14 attributes. It uses three methods to achieve this: a multi-line chart, parallel coordinates and a radar chart. In addition it uses DBSCAN to cluster the songs according to how similar they are. Implemented as a web application using HTML/CSS/JavaScript and the JavaScript library D3.js.

Neural Style Transfer

Uses deep learning to compose images in the style of another image. Neural style transfer is an optimization technique used to take a content image and a style image and blend them together to get the content image but "painted" in the style of the style image. Uses a convolutional neural network to extract features from certain layers and then calculate and iteratively minimize a loss function. Implemented in Python using Tensorflow.

VR card game - Bachelor's project

A multiplayer card game in VR. Two players face of against eachother and controls a golem by playing cards. The goal of the project was to make the game challenging cognitive wise by giving the player alot of options and plenty of things to micro-manage. The game was implemented in Unreal Engine 4 and Oculus Rift as the vr-headsets. Another important part of the project was to practice agile software development in the form of Scrum.

Food review classification

Sentiment analysis of Amazon Fine Food reviews using machinelearning. Uses natural language processing and text analysis to classify if the reviews are positive or negative from only looking at the actual text. Utilizes TF-IDF to score the words according to how important they are and then applies Logistic regression as the learning method. Written in Python.
